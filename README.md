## 🖥️ What termmind does

termmind is a terminal assistant that helps you work from the command line with less effort. It can answer questions, explain commands, help with code, and guide you through common tasks.

It works with several AI providers, including OpenAI, Claude, Gemini, and local models. It also supports 40+ commands for daily terminal work, code review, and quick help.

## ❓ Troubleshooting

### The app will not open

- Check that the download finished
- Extract the files if the download came as a .zip
- Try running the app again
- Right-click the app and choose Run as administrator if needed

### Windows blocks the file

- Re-download from the release page
- Make sure you got the file from the official release link
- Open the file again after the download completes

### My API key does not work

- Check that the key is copied with no extra spaces
- Make sure you picked the correct provider
- Confirm that the provider account is active

### I do not know which file to download

- Open the latest release
- Look for the file made for Windows
- Choose the file that matches your system, if there is more than one
